This website serves as a minimalist online home for the Jobling family, transforming into a vibrant, automated photo gallery that celebrates York, England. Every day, the platform quietly scans public archives to gather a fresh, unique collection of dozens of photographs featuring local landmarks like York Minster and The Shambles, saving them so the page always loads instantly for visitors. Designed to be fast and completely self-sustaining, the site handles everything automatically in the background, requiring no maintenance and using reliable fallback images if the external archives are ever temporarily down.
